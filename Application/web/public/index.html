Tentang Android :

Android adalah Sebuah Sistem Operasi Berbasis Linux Dengan Kode Sumber Terbuka Dan Berlisensi APACHE 2.0 Yang Dirancang Beragam Untuk Perangkat Bergerak Layar Sentuh Seperti Telepon Pintar Dan Komputer Tablet.
Android Awalnya Dikembangkan Oleh Android.Inc.Dengan Dukungan Finansial Dari Google. Yang Kemudian Membelinya Pada Tahun 2005.Sistem Operasi Ini Dirilis Secara Resmi Pada Tahun 2007, Bersamaan Dengan DiDirikannya Open Handset Alliance, Konsorsium Dari Perusahaan-perusahaan Perangkat Keras, Perangkat Lunak, Dan Telekomunikasi Yang Bertujuan Untuk Memajukan Standar Terbuka Perangkat Seluler.
Ponsel Android Pertama Mulai Dijual Pada Bulan Oktober 2008.

Selanjutnya Tentang Android.

Tentang Android Package ?

Android Package Adalah Sebuah Perangkat Lunak Yang Dirancang Untuk Dijalankan Pada Perangkat Android. Istilah Ini Juga Mengacu Pada File APK Yang Merupakan Singkatan Dari Android Package. File Ini Adalah Arsip Zip Yang Berisi Kode Aplikasi, Sumber daya, Dan Informasi Meta. Aplikasi Android Dapat Ditulis Dalam Kotlin, Java, dan C++ Dan Dijalankan Di Dalam Mesin Virtual. Lingkungan Pengembangan Resmi Adalah Android Studio.

Dan Yang Akan Dibahas Disini Adalah Tentang Bagaimana Cara Membuat Android Package Langsung Di SmartPhone Berbasis Android Dengan Menggunakan Aplikasi Aide Atau AndroidIde.

Tentang Aide

Menu

Aide dalah sebuah lingkungan pengembangan terpadu (IDE) untuk mengembangkan aplikasi Android secara langsung pada perangkat Android kamu. Mengikuti pelajaran coding interaktif dan langkah demi langkah (step by step) menjadi pengembang aplikasi yang handal. Aplikasi desain secara visual, menulis kode dengan fitur editor yang kaya dengan listingan, memeriksa kesalahan secara real time, refactoring dan navigasi smart code, menjalankan aplikasi kamu dengan satu klik dan menggunakan debugger Java untuk menemukan bug.

Fitur pelajaran yang interaktif dengan petunjuk langkah demi langkah (step by step) untuk belajar pengembangan aplikasi Android dan keterampilan pemrograman Java. Mengikuti pelajaran, mencoba menjadi ahli coding dan menerapkan apa yang kamu pelajari dalam proyek aplikasi kamu sendiri. AIDE akan mengubah tablet Android kamu dengan keyboard ke dalam "Grid" pembangunan aplikasi yang nyata. Sang Pengembang Aplikasi ini menggunakan Transformer Prime kode dengan AIDE.
